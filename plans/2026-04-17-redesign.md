# Portfolio Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild the home page, blog index, and blog post template as a quiet, senior, principal-engineer register (Literata italic × Atkinson Hyperlegible × Cascadia Code on warm cream paper with an ink-navy accent), adding build-time OG image generation and a new favicon.

**Architecture:** Pure Astro + Tailwind 4 static site. Single centered column (580px max), three content sections per the spec, colophon footer. Build-time OG images generated via `satori` + `@resvg/resvg-js` at a prerendered endpoint (`/og/[...slug].png`). Everything else is static Astro with plain CSS in `global.css` and Tailwind `@theme` tokens for colour + font variables. No JS framework; motion limited to one CSS page-load fade.

**Tech Stack:** Astro 5, Tailwind 4, MDX, Bun. New dependencies: `satori`, `@resvg/resvg-js`, `@fontsource/literata`, `@fontsource/cascadia-code`. Fonts loaded at runtime from Google Fonts; the two TTFs from the `@fontsource` packages are read at build time by satori only.

**Source of truth:** `specs/2026-04-17-redesign-design.md`.

## Task 4: Create the OG endpoint at `src/pages/og/[...slug].png.ts`

Prerendered Astro endpoint. `getStaticPaths()` enumerates home + every post slug. The GET handler calls satori → resvg and returns a PNG. Output lands in `docs/og/*.png` at build.

**Files:**
- Create: `src/pages/og/[...slug].png.ts`

- [ ] **Step 1: Create the endpoint**

```ts
// src/pages/og/[...slug].png.ts
// Prerendered OG PNGs. One per route: index + each blog post.

import type { APIRoute } from "astro";
import { getCollection } from "astro:content";
import satori from "satori";
import { Resvg } from "@resvg/resvg-js";
import { readFileSync } from "node:fs";
import { join } from "node:path";
import { renderOg } from "../../lib/og";

export const prerender = true;

const LITERATA_ITALIC_PATH = join(
    process.cwd(),
    "node_modules/@fontsource/literata/files/literata-latin-400-italic.ttf",
);
const CASCADIA_PATH = join(
    process.cwd(),
    "node_modules/@fontsource/cascadia-code/files/cascadia-code-latin-400-normal.ttf",
);

const literataItalic = readFileSync(LITERATA_ITALIC_PATH);
const cascadia = readFileSync(CASCADIA_PATH);

function formatPubDate(d: Date): string {
    return d.toLocaleDateString("en-GB", {
        day: "numeric",
        month: "long",
        year: "numeric",
    });
}

export async function getStaticPaths() {
    const posts = await getCollection("blog");

    return [
        {
            params: { slug: "index" },
            props: {
                title: "Vaibhav Acharya, engineer.",
                meta: "vaibhavacharya.github.io",
            },
        },
        ...posts.map((post) => ({
            params: { slug: post.id },
            props: {
                title: post.data.title + ".",
                meta: `${formatPubDate(post.data.pubDate)}  ·  vaibhavacharya.github.io`,
            },
        })),
    ];
}

export const GET: APIRoute = async ({ props }) => {
    const { title, meta } = props as { title: string; meta: string };

    const svg = await satori(renderOg({ title, meta }) as any, {
        width: 1200,
        height: 630,
        fonts: [
            {
                name: "Literata",
                data: literataItalic,
                weight: 400,
                style: "italic",
            },
            {
                name: "Cascadia Code",
                data: cascadia,
                weight: 400,
                style: "normal",
            },
        ],
    });

    const png = new Resvg(svg).render().asPng();

    return new Response(png, {
        headers: { "Content-Type": "image/png" },
    });
};
```
